# Create Yanvex App

Create modern React applications with Vite, Tailwind CSS v4, Authentication, Protected Routes, Public Routes, and optional shadcn/ui support.

## Quick Start

```bash
npx create-yanvex-starter@latest my-app
```

or

```bash
npm create create yanvex-starter@latest my-app
```

---

## Features

* React + Vite
* JavaScript or TypeScript
* Tailwind CSS v4
* Public Routes
* Protected Routes
* Authentication Context
* Home Page
* Login Page
* Dashboard Page
* Optional shadcn/ui setup
* Interactive CLI using arrow-key navigation

---

## Usage

```bash
npx create-yanvex-starter@latest
```

The CLI will guide you through:

* Project Name
* Framework Selection
* JavaScript / TypeScript
* shadcn/ui Setup
* Additional Features

---

## Requirements

* Node.js 20+
